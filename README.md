## Installation

1. Clone the repository:

```bash
git clone https://github.com/2Mihye/web-practice
```

2. Change into the project directory:

```bash
#backend
cd server
```

```bash
#frontend
cd client
```

두 폴더 안에서 3번을 각 1회씩 터미널에 입력후 실행 3. Install dependencies:

```bash
npm install
```

## Running the Application

실행하기

```bash
npm start
```
